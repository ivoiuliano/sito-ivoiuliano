# Gallery Images

This directory should contain high-quality images of instruments crafted by Ivo Iuliano.

## Required Images

The gallery component expects the following images (or similar):

1. `violin-1.jpg` - Violin front view
2. `violin-2.jpg` - Violin detail/side view
3. `violin-3.jpg` - Another violin
4. `viola-1.jpg` - Viola front view
5. `viola-2.jpg` - Viola detail
6. `cello-1.jpg` - Cello full view
7. `workshop-1.jpg` - Workshop/atelier view
8. `instrument-detail-1.jpg` - Close-up detail (scroll, f-holes, etc.)
9. `baroque-1.jpg` - Baroque instrument (viola da gamba or similar)

## Image Specifications

- **Format**: JPG or WebP preferred
- **Size**: At least 1200x1200px (square aspect ratio)
- **Quality**: High resolution for professional presentation
- **Style**: Professional photography with good lighting

## B&W Effect

All images will be displayed in grayscale by default and transition to color on hover with a subtle zoom effect.

## Notes

- Images should showcase the craftsmanship and detail of the instruments
- Include variety: full instruments, details, workshop scenes
- Ensure good lighting and focus
- Consider hiring a professional photographer for best results
