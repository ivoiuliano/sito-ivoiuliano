## Sottotitolo

Puoi usare titoli, paragrafi e immagini. Per aggiungere un'immagine, mettila in `public/images/blog/` e scrivi:

![Descrizione immagine](/images/blog/nome-file.webp)

Salva questo file con un nome univoco (es. `come-costruisco-un-violino.md`). Il nome del file (senza .md) diventa l'indirizzo dell'articolo: `/blog/nome-file`. Per la versione in inglese crea lo stesso file in `content/blog/en/` con contenuto tradotto.
